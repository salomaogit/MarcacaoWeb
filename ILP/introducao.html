O que é linguagem de programação?

Linguagem de programação é um meio de comunicação entre o homem (programador) e o computador.

Auxiliam ao programador no processo de descrever para a maquina instruções em uma linguagem mais humana e natural do que a linguagem de máquina, ou seja linguagem binária (0 e 1).

Uma Linguagem de Programação é destinada para ser usada por uma pessoa para expressar um processo através do qual o computador possa resolver um problema.

Computadores só entendem linguagem de máquina (não adequadas para o entendimento humano).

Para executar a finalidade para a qual foram criadas, os programas precisam ser convertidos (traduzidos) para a linguagem de máquina.

Este trabalho é feito por programas intermediários (compiladores e interpretadores).

como é mais fácil de entender/programar?

Para entender você precisa estudar e praticar alguma linguagem

Por que etudar linguagens de programação

Aumento da capacidade de expressar idéias

A capacidade intelectual é influenciada pelo poder de se expressar na linguagem com a qual nos comunicamos.

As linguagens de programação impõem quanto aos tipos de estruturas de dados e abstrações, limitando também as formas de construir algoritmos com ela

A melhor compreensão das linguagens de programação e das suas estruturas ajudam a extrair o máximo de sua funcionalidade e eficiência. Ao expressar soluções para os problemas que vamos resolver ao escrever algorítimos.

Sua lógica e sua capacidade de criar soluções para problemas computacionais melhoram.

Maior embasamento na escolha da linguagem apropriada.

Programadores com pouco conhecimento em linguagem de programação ou programadores muito antigos que não se atualizaram temem a ter problemas para decidir a linguagem mais adequadaao tipo de problema que precisa resolver.

Conhecer várias linguagens de programação (e estar sem atualização) ajuda a fazer escolhas mais consciêntes na hora de implementar a solução para um problema.

Exemplo:

Criar um programa para web com Linguagem C é possível mas é complicado e inviável. Pois existem linguagens mais adequadas para este fim como PHP e JAVASCRIPT
