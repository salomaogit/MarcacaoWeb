Tipos de Linguagens de Programação


Domínio de Programação - Científicas


  • Década de 40
  • Foco de desempenho
  • Linguagem de Montagem (Assembly)
  • Aplicações de Estruturas Simples, mas exigem grande número de comparações aritméticas
  • Estruturas comuns: Vetores e Matrizes
  • Estruturas de controle: Laços de contagens e seleções
  • Primeiras Linguagens desse domínio assembly
    • FORTRAN
    • ALGOL60

Domínio de Programação - Comercial


  • Década de 50
  • Focos em produção de relatórios elaborados
  • Possui métodos precisos para descrever e armazenar números decimais e textos, e capacidade de expressar operações aritméticas decimais
  • Primeira linguagem comercial
    • COBOL 1960
  • Evolução:
    • Planilhas eletrônicas, sistemas gerenciadores de banco de dados, ferramentas de escritório, etc.

Domínio de Programação - Sistemas


  • Software básicos, com S.O., deve possuir eficiência na execução de programas e permitir execução de outros aplicativos.
  • Devem oferecer execuções rápidas e ter recursos de baixo nível que permitam o software fazer interfaces com dispositivos externos.
  • O sistema operacional UNIX foi escrito em linguagem C, tornando fácil portá-lo para diferentes máquinas.
  • A linguagem C é uma linguagem de alto nível com alguns aspectos de linguagem de baixo nível e possui execução eficiente e leve.

Domínio de programação - Script


  • Década de 80
  • Lista de comandos (script) em um arquivo para serem executados
  • Primeira Linguagem: sh (shell), iniciou-se como uma pequena seleção de comandos interpretados como chamada aos subprogramas do sistema que executavam funções como gerenciamento de arquivos, etc
  • Nesta linguagem é assumida a existência de conjunto de componentes já desenvolvidos em outras linguagens, seu objetivo (inicial) é combinar componentes, e não desenvolver programas a partir de estruturas de dados elementares
  • Muito usada para implementar aplicações Web
  • Exemplos:
    • Javascript, Pearl, Pyton, PHP

Propósitos especiais


  • São linguagens criadas para oferecer recursos de computação para hardware de uso específico (máquinas da indústria, calculadores científicos, etc).
  • Exemplos desse tipo de linguagem:
    • RPL(Calculadora HP)
    • RPG(Relatórios comerciais)
    • GPSS(Simulação de sistemas)
